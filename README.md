# coding style
1. 변수명은 소문자로 시작, 카멜 방식 : nameTable, ageGroup
1.1 const 변수명은 대문자와_ 로 사용 : MAX_COUNT
2. 함수명은 대문자로 시작, 동사+명사형태 : SetName()
3. 파일명은 소문자와_ 사용 : ssd_write_program.cpp
4. 중괄호는 함수 바로 옆에 시작
'
   void SetName(string str) {
   ...
   }
'
